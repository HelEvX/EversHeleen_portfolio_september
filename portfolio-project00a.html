Dashboard Project en Portfolio Ontwerp

Werkplekleren 1 Groepsopdracht

Dashboard

Dashboards zijn een unieke en krachtige manier om relevante, bruikbare gegevens te weergeven voor een bepaalde type taak. Ze maken de meest relevante informatie in een oogopslag begrijpelijk en zijn gericht op realtime updates.

Opdracht: ontwerp een dashboard CV waarmee je jezelf op de markt zet in drie aparte onderdelen die samen één digitaal geheel (website) vormen:

  • Dashboard per persoon (inhoud individueel, opmaak via groepswerk)
  • Project traject pagina (documentatie en tijdslijn, inhoud door groep)
  • Portfolio pagina (individueel, zelf alles te kiezen, wireframe voor volgende opdracht)

Portfolio

Zet het visueel design van je portfolio om naar één of meerdere werkende webpagina. Voorzie minstens volgende inhoud op de portfolio-webpagina:

  • Korte voorstelling (Wie ben jij? Wat maakt je anders/speciaal?)
  • (School)projecten per categorie
  • Opdrachten Werkplekleren 1
  • Eindreflectie Werkplekleren 1

Deze opdracht heb ik laten verschuiven naar de tweede zittijd door tijdgebrek. Ik maakte daarvoor een volledig nieuw design (naar projectpagina).
